Параметры

Настройка объединения теперь делается прямо в интерфейсе. JSON через UI больше не нужен.
Группы вывода
Каждая группа задаёт логический id и папку внутри output, куда будут складываться победившие файлы.
Правила сопоставления аддонов
Правила проверяются сверху вниз. Можно матчить точное имя папки аддона или regex-паттерн.
Fallback для нераспознанных аддонов
Определи, что делать с аддонами, для которых не сработало ни одно правило.
Дополнительная пост-обработка
Если путь пустой, инструмент попробует env, проектную папку bin/, а затем системный PATH.
Параметры деления
Папки не разрезаются — если папка не вписывается в текущую часть, она целиком переходит в следующую. Части именуются 01_core_part1, 01_core_part2 и т.д.
Браузерный UI работает через localhost, а helper-сервер получает доступ к локальной файловой системе. Инструмент рассчитан на доверенное локальное использование.
